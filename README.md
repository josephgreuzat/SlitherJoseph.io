# SlitherJoseph.io

Multiplayer snake game similar to Slither.io built with Node.js and Canvas.

## Installation Instructions

Run the following commands:

```bash
npm install
npm start
```

## Usage

Open `http://localhost:8080` in your browser.

## Features
- Real-time multiplayer gameplay with WebSocket
- Solo mode with AI bots
- Leaderboard system
- Smooth Canvas-based rendering
- Room-based matchmaking
